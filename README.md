## Projects

### 1. Optical Camera Communication (OCC) — Prototype System
> **Visible Light Communication using a webcam and LED**

![Python](https://img.shields.io/badge/Python-3.x-blue?style=flat&logo=python)
![OpenCV](https://img.shields.io/badge/OpenCV-Computer%20Vision-green?style=flat&logo=opencv)
![Arduino](https://img.shields.io/badge/Arduino-Uno-teal?style=flat&logo=arduino)

汎用Webカメラ（30fps）とLEDを用いた可視光通信システムのプロトタイプ。2-PAM変調・フレーム同期アルゴリズムを自ら設計し、実環境（50〜150cm）で **BER 0.00%** を達成。

**Key Highlights:**
- Modulation: 2-PAM, Symbol Rate: 15 baud
- Custom synchronization algorithm (edge detection + phase estimation)
- Verified BER 0.00% under controlled conditions

📁 [Repository](./02_Research_OCC/)

## Research (In Progress)
 
### HAP-Ground Station Network for QKD Services in Tohoku
> **Quantum Key Distribution using High-Altitude Platforms**
 
会津大学にて取り組んでいる研究テーマ。成層圏（高度20km）を飛行するHAP（高高度プラットフォーム）を中継ノードとして活用し、東北地域へのQKDサービス提供の実現可能性を評価しています。
 
ERA5気象再解析データを用いて雲による通信遮断を定量化し、複数の光学地上局（OGS）によるサイトダイバーシティの有効性を検証。BB84プロトコルの物理モデルに基づくSKR（秘密鍵生成レート）の推定まで実施。
 
**Key Topics:** Quantum Communications, Free-Space Optics, Site Diversity, BB84, ERA5
 
> ※現在進行中の研究のため、コードの公開は準備中です。
